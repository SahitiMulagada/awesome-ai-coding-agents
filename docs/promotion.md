# Promotion Strategy

A checklist for getting visibility and stars on the `awesome-ai-coding-agents` repository.

## Week 1: Launch

- [ ] Share on Reddit: [r/programming](https://reddit.com/r/programming), [r/artificial](https://reddit.com/r/artificial), [r/coding](https://reddit.com/r/coding), [r/MachineLearning](https://reddit.com/r/MachineLearning)
- [ ] Post on [Hacker News](https://news.ycombinator.com/) (aim for Tuesday-Thursday, 8-10 AM EST for best visibility)
- [ ] Share on X/Twitter with hashtags: `#AI`, `#CodingAgents`, `#DevTools`, `#OpenSource`, `#AwesomeList`
- [ ] Post on [Dev.to](https://dev.to/) as a blog article summarizing the list
- [ ] Share on LinkedIn with a short write-up

## Week 2-4: Build momentum

- [ ] Reach out to tool maintainers listed in the repo (they often reshare and contribute)
- [ ] Cross-post to relevant Discord servers (AI, developer tools, open source communities)
- [ ] Share in relevant Slack communities
- [ ] Submit to [Product Hunt](https://www.producthunt.com/) as a resource
- [ ] Post updates as new tools are added

## After 30 days: Awesome list submission

- [ ] Review 4 other pull requests on [sindresorhus/awesome](https://github.com/sindresorhus/awesome) (required before submitting)
- [ ] Submit a PR to add this list to the official awesome list
- [ ] Submit PRs to related awesome lists (awesome-llm, awesome-developer-tools, etc.)

## Ongoing

- [ ] Monitor GitHub issues and PRs for community contributions
- [ ] Keep the list updated as new tools launch
- [ ] Engage with contributors and thank them publicly
- [ ] Track star count and referral sources via GitHub Insights
