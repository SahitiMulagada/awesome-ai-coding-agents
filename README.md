## CLI Agents

Terminal-based coding agents that work directly from your command line.

- [AdaL](https://github.com/SylphAI-Inc/adal-cli) - Self-evolving coding agent that learns from your team and codebase, built on AdalFlow.
- [AiderDesk](https://github.com/hotovo/aider-desk) - Desktop GUI and orchestration layer for AI-powered coding with project management and Git worktree isolation.
- [Claude Code](https://github.com/anthropics/claude-code) - Anthropic's agentic coding tool that operates in your terminal with full codebase understanding.
- [Crush](https://github.com/charmbracelet/crush) - Charmbracelet's terminal-based AI coding agent with LSP integration and session management.
- [Devin](https://devin.ai/) - Cognition's autonomous software engineer with local-to-cloud handoff for long-running tasks.
- [Droid](https://factory.ai/) - Factory's enterprise-grade terminal coding agent ranked number one on Terminal-Bench with multi-model parallel sessions.
- [ForgeCode](https://github.com/tailcallhq/forgecode) - Open-source AI pair programmer for the terminal supporting 300+ models with custom agent definitions.
- [Goose](https://github.com/aaif-goose/goose) - Open-source autonomous developer agent by the Agentic AI Foundation supporting 70+ MCP extensions.
- [iFlow CLI](https://github.com/iflow-ai/iflow-cli) - Free terminal AI assistant with access to Chinese AI models through the iFlow platform.
- [Junie](https://junie.jetbrains.com/) - JetBrains' LLM-agnostic coding agent for terminal, IDE, and CI/CD pipelines.
- [Kode](https://github.com/shareAI-lab/Kode-cli) - Terminal coding agent with multi-model collaboration and expert model consultation via @ask mentions.
- [Mistral Vibe](https://github.com/mistralai/mistral-vibe) - Mistral's open-source CLI coding agent powered by Devstral models with slash-command skills.
- [Neovate](https://github.com/neovateai/neovate-code) - Terminal-based coding agent supporting 400+ models with multi-agent support and image input.
- [Pi](https://github.com/earendil-works/pi) - Minimal and self-extensible terminal coding harness with only four built-in tools.
- [Qwen Code](https://github.com/QwenLM/qwen-code) - Alibaba's open-source terminal coding agent optimized for Qwen3-Coder with 480B parameters.
- [Rovo Dev](https://www.atlassian.com/software/rovo-dev) - Atlassian's AI coding agent with deep Jira, Confluence, and Bitbucket integration.
- [Tabnine](https://www.tabnine.com/) - Enterprise-grade terminal coding agent with privacy-first architecture and both interactive and autonomous modes.

## IDE Extensions

AI coding agents that plug into your existing code editor.

- [Cline](https://cline.bot/) - Open-source autonomous AI coding agent with Plan/Act workflow and browser automation.
- [Continue](https://continue.dev/) - Open-source AI code assistant for VS Code and JetBrains with customizable model support and MCP integration.
- [Kilo Code](https://kilo.ai/) - Open-source AI coding agent for VS Code and JetBrains with 500+ model support and inline autocomplete.
- [Pochi](https://docs.getpochi.com/) - AI-powered VS Code coding agent by TabbyML with deep GitHub integration and cloud collaboration.
- [Roo Code](https://github.com/RooCodeInc/Roo-Code) - Open-source VS Code extension providing customizable AI modes for architecture, coding, and debugging.
- [Sourcegraph Cody](https://sourcegraph.com/cody) - AI assistant with multi-repository code graph intelligence.
- [Supermaven](https://supermaven.com/) - Ultra-fast AI completions with 1M token context window and sub-250ms response times.
- [Zencoder](https://zencoder.ai/) - Universal AI coding agent platform unifying multiple AI tools inside VS Code and JetBrains.

## IDEs & Code Editors

Full development environments and code editors with AI capabilities.

- [Cursor](https://cursor.com/) - AI-native code editor with Composer for multi-file agentic editing and auto model selection.
- [Emacs](https://www.gnu.org/software/emacs/) - Extensible editor with AI ecosystem including GPTel, Ellama, and copilot.el.
- [JetBrains Air](https://www.jetbrains.com/air/) - Agent-first IDE running multiple AI agents concurrently in isolated workspaces.
- [JetBrains IDEs](https://www.jetbrains.com/ai/) - Professional IDEs with built-in AI Assistant and Junie agent.
- [Kiro](https://kiro.dev/) - AWS's spec-driven agentic IDE built on VS Code with automated user stories and task generation.
- [Neovim](https://neovim.io/) - Terminal editor with AI plugins including avante.nvim and CodeCompanion.nvim.
- [PearAI](https://trypear.ai/) - Open-source AI code editor with unified multi-model subscription routing.
- [Qoder](https://qoder.com/) - Alibaba's agentic IDE with Quest Mode for autonomous task delegation and multi-interface support.
- [Replit](https://replit.com/) - Cloud IDE with AI Agent v3 for full-stack app generation from natural language.
- [Trae](https://trae.ai/) - ByteDance's AI-native IDE with Builder mode for end-to-end feature implementation and multimodal input.
- [Trae CN](https://www.trae.com.cn/) - Chinese edition of Trae IDE powered by Doubao and DeepSeek models with SOLO mode for autonomous development.
- [Visual Studio](https://visualstudio.microsoft.com/) - Microsoft's IDE with deeply integrated Copilot for .NET and C++ development.
- [VS Code](https://code.visualstudio.com/) - Dominant code editor with GitHub Copilot agent mode and MCP support.
- [Windsurf](https://windsurf.com/) - AI-native IDE with Cascade for deep codebase understanding and fluid copilot-to-agent transitions.
- [Xcode](https://developer.apple.com/xcode/) - Apple's IDE with built-in AI coding intelligence and agentic coding support.
- [Zed](https://zed.dev/) - High-performance open-source editor built in Rust with native AI and Agent Client Protocol.

## Enterprise Platforms

AI coding platforms designed for team and organization-wide deployment.

- [Augment](https://www.augmentcode.com/) - Enterprise-scale agentic development platform with multi-model routing via Prism across Claude, GPT, and Gemini.
- [CodeArts Agent](https://www.huaweicloud.com/product/codearts.html) - Huawei Cloud's enterprise AI coding agent covering full DevSecOps lifecycle with HarmonyOS support.
- [CodeBuddy](https://www.codebuddy.ai/) - Tencent Cloud's AI coding assistant with Craft feature for natural-language-to-codebase generation.
- [Cortex Code](https://www.snowflake.com/en/product/features/cortex-code/) - Snowflake-native AI coding agent for data engineering that understands enterprise schemas and governance rules.
- [IBM Bob](https://bob.ibm.com/) - IBM's AI-first development partner covering the full SDLC with multi-model routing and role-based agents.

## Multi-Agent Orchestrators

Frameworks for coordinating multiple AI agents working together on development tasks.

- [Mux](https://github.com/coder/mux) - Open-source desktop app by Coder for running multiple coding agents in parallel with isolated workspaces.

## General-Purpose AI Agents

Versatile AI agents with strong software engineering capabilities among broader skills.

- [Hermes Agent](https://github.com/NousResearch/hermes-agent) - Self-improving AI agent by Nous Research with persistent memory and multi-platform gateway support.
- [OpenClaw](https://github.com/nichochar/openclaw) - Open-source autonomous AI agent that runs locally and connects to Slack, Discord, Telegram, and WhatsApp.
- [OpenHands](https://github.com/All-Hands-AI/OpenHands) - Open-source autonomous AI software engineer that modifies code, executes commands, and browses the web.
