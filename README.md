## CLI Agents

Terminal-based coding agents that work directly from your command line.

- [AdaL](https://github.com/SylphAI-Inc/adal-cli) - Self-evolving coding agent that learns from your team and codebase.
- [AiderDesk](https://github.com/hotovo/aider-desk) - Desktop GUI and orchestration layer for AI-powered coding with project management and Git worktree isolation.
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) - Anthropic's agentic coding tool that operates in your terminal with full codebase understanding.
- [Crush](https://github.com/charmbracelet/mods) - Charmbracelet's terminal-based AI coding agent with a focus on developer experience.
- [Devin](https://devin.ai/) - Cognition's autonomous software engineer available as CLI and web interface.
- [Droid](https://factory.ai/) - Factory's enterprise-grade terminal coding agent supporting multi-model workflows and parallel sessions.
- [ForgeCode](https://forgecode.dev/) - AI coding agent for your terminal with multi-model support and MCP integration.
- [Goose](https://github.com/block/goose) - Block's open-source autonomous developer agent that operates on your machine.
- [iFlow CLI](https://www.iflow.dev/) - AI coding assistant for the terminal, currently sunsetting.
- [Junie](https://junie.jetbrains.com/) - JetBrains' LLM-agnostic coding agent for terminal, IDE, and CI/CD pipelines.
- [Kode](https://kode.dev/) - Terminal-first AI coding assistant with codebase-aware context.
- [Mistral Vibe](https://mistral.ai/) - Mistral AI's coding agent for generating applications from natural language prompts.
- [Neovate](https://github.com/neovate-ai/neovate) - AI-powered Neovim plugin bringing agentic coding directly into your editor.
- [Pi](https://github.com/earendil-works/pi) - Minimal and self-extensible terminal coding harness by Earendil Works.
- [Qwen Code](https://github.com/QwenLM/qwen-agent) - Alibaba's AI coding agent powered by the Qwen model family.
- [Rovo Dev](https://www.atlassian.com/rovo) - Atlassian's AI teammate that integrates with Jira and Bitbucket for development workflows.
- [Tabnine](https://www.tabnine.com/) - AI code assistant with CLI support, running locally or in the cloud with enterprise privacy controls.

## IDE Extensions

AI coding agents that plug into your existing code editor.

- [Continue](https://continue.dev/) - Open-source AI code assistant for VS Code and JetBrains with customizable model support.
- [Kilo Code](https://kilocode.ai/) - VS Code extension with autonomous and interactive coding modes and MCP support.
- [Pochi](https://pochi.dev/) - AI pair programmer extension for VS Code with codebase understanding.
- [Roo Code](https://roocode.com/) - Autonomous coding agent for VS Code with multi-model support and MCP integration.
- [Zencoder](https://zencoder.ai/) - AI coding agent extension for VS Code and JetBrains with repo-wide context awareness.

## AI-Native IDEs

Full development environments built from the ground up around AI capabilities.

- [Kiro](https://kiro.dev/) - AWS's AI-powered IDE built on VS Code with spec-driven development and steering hooks.
- [Qoder](https://qoder.com/) - AI-native IDE with Quest Mode for asynchronous autonomous task execution and MCP integration.
- [Trae](https://trae.ai/) - ByteDance's AI-native IDE with Builder mode for generating applications from natural language.
- [Trae CN](https://trae.com.cn/) - Chinese edition of Trae IDE powered by Doubao and DeepSeek models, optimized for Chinese developers.
- [Windsurf](https://windsurf.com/) - AI-native IDE with Cascade for deep codebase understanding and multi-file editing.

## Enterprise Platforms

AI coding platforms designed for team and organization-wide deployment.

- [Augment](https://www.augmentcode.com/) - Enterprise AI coding platform with deep codebase understanding and context-aware suggestions.
- [CodeArts Agent](https://www.huaweicloud.com/product/codearts.html) - Huawei Cloud's AI-powered software development platform for enterprise teams.
- [CodeBuddy](https://cloud.tencent.com/product/codebuddy) - Tencent Cloud's AI coding assistant for enterprise development workflows.
- [Cortex Code](https://cortex.io/) - AI-powered developer portal with coding assistance and service catalog integration.
- [IBM watsonx Code Assistant](https://www.ibm.com/products/watsonx-code-assistant) - IBM's enterprise AI coding assistant with COBOL-to-Java modernization and Ansible automation.

## Multi-Agent Orchestrators

Frameworks for coordinating multiple AI agents working together on development tasks.

- [Mux](https://github.com/nicobailon/mux) - Multi-agent orchestrator that coordinates specialized AI coding agents working in parallel.

## General-Purpose AI Agents

Versatile AI agents with strong software engineering capabilities among broader skills.

- [Hermes Agent](https://github.com/hermes-agent/hermes) - General-purpose AI agent framework with strong coding and tool-use capabilities.
- [OpenClaw](https://github.com/openclaw-ai/openclaw) - Open-source AI agent platform with code generation and autonomous task execution.
- [OpenHands](https://github.com/All-Hands-AI/OpenHands) - Open-source platform for AI-powered software development agents with a web interface.

## Developer Tooling

Tools that enhance the AI coding agent ecosystem.

- [CodeMaker AI](https://codemaker.ai/) - Automated code generation and documentation tool for IDEs, CLI, and CI/CD workflows.
- [MCPJam](https://mcpjam.com/) - Testing and evaluation platform for debugging, inspecting, and running evals on MCP servers.

## Claude Code Addons

Frameworks, plugins, custom commands, and tools that extend Claude Code's capabilities.

- [Awesome Claude Code](https://github.com/jqueryscript/awesome-claude-code) - Curated list of tools, IDE integrations, frameworks, and resources for Claude Code.
- [Awesome Claude Code Plugins](https://github.com/ccplugins/awesome-claude-code-plugins) - Curated list of slash commands, subagents, MCP servers, and hooks for Claude Code.
- [Awesome Claude Plugins](https://github.com/ComposioHQ/awesome-claude-plugins) - Curated list of plugins that extend Claude Code with custom commands, agents, and hooks.
- [BMAD-AT-CLAUDE](https://github.com/24601/BMAD-AT-CLAUDE) - Full-stack AI agent framework for Claude Code with structured methodology and agent personas.
- [Claude Code Guide](https://github.com/Cranot/claude-code-guide) - Comprehensive auto-updated guide documenting Claude Code CLI features, tools, and workflows.
- [Claude Code System Prompts](https://github.com/Piebald-AI/claude-code-system-prompts) - Documented collection of all Claude Code system prompt parts, tool descriptions, and sub-agent prompts.
- [Claude Context](https://github.com/zilliztech/claude-context) - Code search MCP server that makes your entire codebase available as context for Claude Code.
- [Claude MPM](https://github.com/bobmatnyc/claude-mpm) - Multi-agent project manager framework with 47+ specialized agents and session management.
- [Context Mode](https://github.com/mksglu/context-mode) - Context window optimization plugin achieving 98% reduction in context consumption across 15 platforms.
- [ContextGraph](https://github.com/contextgraph/claude-code-plugin) - AI-forward persistent task management plugin for Claude Code.
- [Superpowers](https://github.com/obra/superpowers) - Collection of custom slash commands and extensions that enhance Claude Code workflows.
- [Taskmaster](https://github.com/eyaltoledano/claude-task-master) - AI-powered task management system with MCP integration for structuring development workflows.
